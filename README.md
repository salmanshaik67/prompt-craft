**PromptCraft** is a futuristic, web-based tool designed to help users engineer high-quality prompts for AI models. Built with a "Cyberpunk/Sci-Fi" aesthetic, it transforms raw ideas into detailed, structured prompts suitable for LLMs and Image Generators.

## üöÄ Features

*   **Futuristic UI/UX**: Immersive glassmorphism design with neon accents, floating 3D orbs, and animated backgrounds.
*   **Smart Enhancement**: Automatically structures prompts with role-playing (e.g., "Act as...") and adds high-fidelity keywords (e.g., "8k resolution", "cinematic lighting").
*   **Instant Feedback**: "Glitch" text effects and typewriter animations for a dynamic user experience.
*   **One-Click Copy**: Easily copy the crafted prompt to your clipboard.

## üì¶ Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/salmanshaik67/prompt-craft.git
    cd prompt-craft
    ```

2.  **Create a virtual environment (Optional but recommended)**
    ```bash
    python -m venv venv
    # Windows
    venv\Scripts\activate
    # macOS/Linux
    source venv/bin/activate
    ```

3.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```
